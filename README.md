# AI Transportation Frontend Slides

Editable HTML slide deck for a 15-minute presentation on AI as a task-oriented assistant for transportation planning and traffic engineering professionals.

## Contents

- `index.html` - self-contained frontend-slides editable deck
- `assets/` - generated comic-style illustrations sized to match the deck frames

## View Locally

Open `index.html` directly in a browser.

## Deck Features

- Neo-Grid Bold visual system adapted to AtkinsRealis-inspired colors
- Editable slide runtime with Pages, Edit, and Export HTML controls
- Comic-style transportation illustrations generated in frame-matched ratios
- Staggered slide entrance animations
- Animated dummy chart loading indicators
- Reduced-motion support

## Editing

Hover the top-left corner in the deck to reveal the editing controls. Use `Pages` for slide thumbnails and `Edit` to adjust slide content.
